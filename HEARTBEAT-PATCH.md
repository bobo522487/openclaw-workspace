## 每日早报逻辑

如果收到系统事件消息：
- `[DAILY_MORNING_REPORT]` - 每日早报

执行操作：
1. 获取天气信息
2. 搜索科技新闻（3条）
3. 搜索AI新闻（3条）
4. 生成并发送早报消息

## 技术说明

### Cron 任务配置
- **任务名称：** 每日早报
- **Cron 表达式：** `15 0 * * *` (每天 UTC 00:15)
- **时区：** Asia/Shanghai (UTC+8)
- **实际执行时间：** 每天 08:15 北京时间

### 执行流程
1. Cron 任务在 08:15 触发
2. 发送消息 `[DAILY_MORNING_REPORT]` 给 agent
3. Agent 收到消息后，HEARTBEAT.md 中的逻辑生效
4. Agent 执行：
   - 获取贵阳天气
   - 搜索科技新闻（3条）
   - 搜索AI新闻（3条）
   - 生成早报消息
   - 发送到 Telegram
