## 🔑 配置步骤

### 1. 获取Home Assistant Token

请主人按照以下步骤操作：

1. 打开Home Assistant：http://10.10.10.8:8123
2. 点击左下角用户头像 → **个人资料页面**
3. 滚动到底部 → 点击 **Long-Lived Access Tokens** → **Create Token**
4. 输入名称（如："露娜"）→ 生成后复制Token
5. 将Token填入文件 `smart-home/ha_token.txt`：
   ```
   TOKEN=your_token_here
   ```

### 2. 测试连接

配置Token后，告诉露娜："测试Home Assistant连接"，露娜会自动：
- 验证连接状态
- 获取所有设备列表
- 更新设备配置文件
